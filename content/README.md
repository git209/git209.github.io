# V6 content workflow

Update the site without changing page structure:

- `data/v6/current-systems.json`: the three systems shown on the home page.
- `data/v6/project-stories.json`: problem, approach, evidence, and next step for each featured project.
- `data/v6/international.json`: hosted visitors and exchange stories.
- `data/publications.json`: selected publications only; use DOI links whenever available.

When adding an item, update the JSON text and the referenced image in `assets/`. Keep submitted or in-preparation work clearly labelled.
